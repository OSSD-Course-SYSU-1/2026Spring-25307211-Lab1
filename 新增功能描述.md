# 第三次提交：联系人分组筛选拓展功能说明
## 1. 新增功能概述
在原有分布式通讯录基础上，实现联系人分组标签管理，分组包含：全部、家人、朋友、同事。
1. 新增联系人时可选择所属分组；
2. 首页顶部下拉框可按分组筛选联系人；
3. 编辑页面自动回显原有分组；
4. 联系人列表、详情页展示分组标签；
5. 分组数据存入分布式数据库，支持跨设备同步。

## 2. 修改文件清单
1. ContactViewModel.ets：ContactData实体新增groupName分组字段
2. ContactListItem.ets：列表条目展示分组标签
3. ContactHomePage.ets：增加分组Picker下拉筛选器
4. ContactAddAndEditPage.ets：新增/编辑页添加分组选择框
5. ContactDetailPage.ets：详情页展示分组信息
6. KvManager.ets：新增按分组查询联系人的数据库方法
